Este software permite el registro de un cultivo y posteriormente
el registro de predios o terrenos que en determinado a√±o tienen sembrado ese cultivo,
la seleccion del predio es a traves de un mapa de la region que permite marcar un

<br />
Esto es realizado utilizando la API de Google Maps.
<br />
El administrador lleva a cabo el registro de los predios.
Cualquier persona se puede registrar en el sistema y realizar consultas por cultivo, estado o municipio observando los predios registrados en el mapa 
con alguno datos adicionales como la cantidad de hectareas cultivadas en cada uno de los casos.
<br /><br />
Este proyecto fue desarrollado utilizando para el frontend "React con Vite" y para el backend "Nestjs"
